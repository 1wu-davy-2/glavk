# glavk Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver a Docker-ready Chinese Web-system credential manager under `glavk`.

**Architecture:** Build a new React/Vite frontend and FastAPI backend in `glavk`, backed by MariaDB `glavk`. Keep authentication, project CRUD, credential encryption, and frontend session handling in focused modules. Use nginx to serve the compiled SPA and proxy `/api` to the backend.

**Tech Stack:** React 18, TypeScript, Vite, Vitest, Testing Library, lucide-react, Python 3.12, FastAPI, SQLAlchemy, PyMySQL, cryptography, pytest, MariaDB 11, Docker Compose.

**Status:** Complete. Fresh backend/frontend tests, build, smoke test, and browser verification passed. Docker files are ready; Compose execution is blocked only by the missing Docker CLI.
