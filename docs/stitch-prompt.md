# Stitch UI 生成话术

请设计一个中文后台管理系统，产品名称为“glavk”，用于管理多个网页 Web 系统。

整体风格：

- 采用蓝白色系，清爽、专业、克制，适合长期使用的内部运营工具。
- 主色为明亮企业蓝，背景为非常浅的雾蓝白，内容区域使用纯白。
- 使用深蓝灰文字、浅蓝边框，使用青绿色表示正常，橙色表示提醒，红色表示删除或异常。
- 不要使用紫色渐变、夸张插画、营销型 Hero 区域、过度圆角或装饰性大卡片。
- 所有页面文字必须使用简体中文。
- 页面需要有明确层级、留白和信息密度，适合快速浏览和重复操作。
- 卡片圆角控制在 8px 左右，阴影轻微，整体像成熟的 SaaS 管理后台。

请生成以下页面：

1. 登录页

- 标题“登录 glavk”。
- 副标题“集中管理你的网页系统入口与登录凭据”。
- 管理员账号输入框、登录密码输入框、显示/隐藏密码按钮、登录按钮。
- 登录失败提示、登录中状态。
- 页面居中白色面板，蓝色品牌图形标识，底部显示“安全连接 · 管理员入口”。

2. 网页系统首页

- 左侧固定导航，宽度约 230px。
- 品牌名称“glavk”，副标题“网页系统管理”。
- 导航菜单：网页系统、收藏项目、设置。
- 底部显示管理员头像、管理员名称和退出登录。
- 顶部显示标题“网页系统”、描述“集中管理所有业务入口与登录凭据”、搜索框“搜索系统名称或地址”、分类筛选和“添加系统”按钮。
- 首屏显示三个概览：全部系统、正常运行、我的收藏。
- 主区域使用三列响应式网格展示 Web 系统卡片。

3. Web 系统卡片

- 彩色方形系统图标或首字母图标。
- 系统名称、分类、正常/已停用状态、访问地址、登录用户名。
- 登录密码默认显示为圆点掩码，仅提供复制密码图标；点击复制时才在浏览器内存中解密。
- 项目如果有截图，卡片顶部展示网页首屏截图；没有截图时使用首字母图标。
- 收藏按钮、最近更新时间、打开系统、编辑和删除操作。
- 打开系统使用新标签页打开访问地址。
- 悬停时边框变为蓝色，并出现轻微上浮效果。

4. 添加/编辑系统抽屉

- 从右侧滑出，桌面端宽度约 440px，手机端全屏。
- 字段：系统名称、访问地址、系统分类、登录用户名、系统密码、项目简介、备注、是否收藏、是否启用。
- 密码字段有显示/隐藏按钮，保存时用户名和密码均非必填。
- 底部固定“取消”和“保存系统”按钮。
- 需要必填校验、错误提示和保存中状态。

5. 反馈状态

- 空状态文字“还没有网页系统”，按钮“添加第一个系统”。
- 加载时显示卡片骨架屏。
- 网络错误显示“暂时无法加载系统列表”。
- 删除前显示确认弹窗。
- 保存成功、删除成功、复制成功显示右下角轻提示。

6. 响应式

- 桌面端左侧导航 + 三列卡片。
- 中等屏幕两列卡片。
- 手机端顶部导航 + 单列卡片。
- 所有按钮、表单和文字不能互相遮挡或溢出。
- 图标使用简洁的线性图标风格。
